实例一

> 输入："(()())(())"
输出："()()()"
解释：
输入字符串为 "(()())(())"，原语化分解得到 "(()())" + "(())"，
删除每个部分中的最外层括号后得到 "()()" + "()" = "()()()"。
```
    var removeOuterParentheses = function (S) {
      let result = '';
      let count = 0;
      let start = 0;
      for (let i = 0; i < S.length; i++) {
        if (S[i] === '(') {
          count++
        } else {
          if (--count === 0) {
            result += S.substring(start + 1, i)
            start = i + 1
          }
        }
      }
      return result;
    };
```
